Timer and Lock Flow

Step Trigger What happens Timer state Tray icon
1. App starts Guardian launched Waits, polling for browser every 2s IDLE 🟢 Green
2. Browser opens Chrome or Edge detected 30-min countdown starts RUNNING 🟢 Green
3. Son closes browser No browser process found Timer pauses — remaining time is banked PAUSED 🟠 Orange
4a. Browser reopened — time left Browser detected, PAUSED state Timer resumes from where it stopped RUNNING 🟢 Green
4b. Browser reopened — no time left Browser detected, EXPIRED state Password dialog shown immediately EXPIRED 🔴 Red
5. Timer hits zero Countdown reaches 0 (browser open) Password dialog shown with 30s countdown EXPIRED 🔴 Red
6a. Correct password entered Parent types right password New full 30-min session starts RUNNING 🟢 Green
6b. Wrong password or timeout Wrong password OR 30s elapses Chrome + Edge killed. Timer stays EXPIRED. Never auto-restarts. EXPIRED 🔴 Red
7. Son tries to reopen browser Browser detected, EXPIRED state Password dialog shown immediately again EXPIRED 🔴 Red
Key rule: The timer NEVER auto-resets. The only way to get a new session is for a parent to enter the correct password. Closing the browser to avoid the dialog doesn't help — the timer stays expired.

Dashboard Flow

StepWhat happens
1. First-time setup Dashboard asks for Supabase URL and Anon Key. Stored in browser localStorage — only needed once.
2. Pick a date and click Load Two API calls run in parallel to Supabase REST API.
3. Domain summary All domain names for the day are fetched, counted and sorted. Top 12 shown as chips with visit counts.
4. History table 50 rows loaded (newest first). Each row shows: time, domain, page title (clickable link), safety badge.
5. Colour coding HIGH Red row — adult content, gambling
MEDIUM Orange row — gaming, social media
LOW Yellow row — borderline
SAFE Normal row — safe sites
6. Flagged count Red warning shown in toolbar if any flagged rows exist (e.g. "⚠️ 3 flagged").
7. Pagination If more than 50 rows exist, Next / Previous buttons appear.
No server needed: The dashboard is a plain HTML file that talks directly to Supabase's REST API from your browser. Open it from your phone, laptop, or anywhere — no login, no app install needed.
